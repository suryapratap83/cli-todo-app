# CLI To-Do App 📝

A command-line to-do app built in Python to manage your daily tasks. Tasks are saved to a file so they persist between sessions.

## Features
- Add single or multiple tasks
- View all tasks with done/pending status
- Mark tasks as done
- Delete tasks
- Search task by number
- Input validation throughout
- Auto-saves tasks to a file

## How to Run

Make sure Python is installed, then run:

python todo.py
